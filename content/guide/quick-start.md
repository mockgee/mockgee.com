  The quickest way to get started with emock is to to follow below steps:

```bash
$ git clone https://github.com/mockgee/mockgee.git
```

  Create the app:

```bash
$ cd mockgee
```

  Install dependencies:

```bash
$ npm install
```

  Start the server:

```bash
$ npm start
```
